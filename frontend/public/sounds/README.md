# Notification Sounds

This directory should contain notification sound files in MP3 format.

Required files:
- bell.mp3
- chime.mp3
- digital.mp3
- kitchen.mp3
- soft.mp3
- wood.mp3

You can add your own sound files or download free sounds from:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://pixabay.com/sound-effects/

Make sure the sounds are short (1-3 seconds) and not too loud.
